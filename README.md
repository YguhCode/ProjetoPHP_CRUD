# 📚 Sistema de Cadastro de Alunos (PHP)

Projeto simples desenvolvido com **PHP + HTML + CSS**, com o objetivo de praticar conceitos de:

* CRUD (Create, Read, Update, Delete)
* Manipulação de arquivos
* Estruturação de layout com CSS
* Boas práticas de organização de código

---

## 🚀 Funcionalidades

✅ Cadastro de alunos
✅ Listagem de alunos
✅ Edição de dados
✅ Exclusão com confirmação
✅ Busca por nome
✅ Cálculo automático de média
✅ Exibição de situação (Aprovado/Reprovado) com cores

---

## 🖥️ Tecnologias utilizadas

* PHP
* HTML5
* CSS3

---

## 📂 Estrutura do projeto

```bash
📁 projeto/
│
├── index.php        # Cadastro de alunos
├── lista.php        # Listagem + busca
├── editar.php       # Edição de alunos
├── atualizar.php    # Processa edição
├── excluir.php      # Remove aluno
├── processa.php     # Processa cadastro
│
├── style.css        # Estilos do sistema
│
└── inc/
    └── funcoes.php  # Funções auxiliares
```
